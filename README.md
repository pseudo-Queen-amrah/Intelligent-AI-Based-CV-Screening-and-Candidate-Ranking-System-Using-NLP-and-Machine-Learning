# Intelligent AI-Based CV Screening and Candidate
Ranking System Using NLP and Machine
Learning

## Overview

This project is an AI-based CV screening system that uses NLP and Machine Learning techniques to evaluate resumes and match them with job descriptions.

## Features

* CV text analysis
* Skill matching using TF-IDF
* Candidate ranking system
* Simple GUI interface

## Technologies Used

* Python
* Scikit-learn
* Tkinter

## How it works

1. Upload CV
2. System extracts text
3. AI model calculates similarity score
4. Final ranking is generated
